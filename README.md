### <u>Deployed Link</u>


Deployed with Netlify 
```
https://lone.netlify.app/
 ```

### <u>Run Locally</u>

Clone the project

```
git clone https://github.com/abhishekmardiya/shop.com-clone.git
```

Install npm packages

```
npm install
```

Run the surver

```
npm run start
```

View in Browser

```
go to http://localhost:3000
```
